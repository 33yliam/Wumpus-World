## Getting Started
 <img src="/images/preview.png" alt="Logo" width="100%">

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/33yliam/Wumpus-World.git
   ```
2. Install packages
   ```sh
    pip install -r requirements.txt
   ```
3. Run the program
   ```sh
   python main.py
   ```
